# Design System & Aesthetics

## Core Philosophy
- **"Enterprise Quality at Startup Speed"**
- **Aesthetic:** Modern, Premium, Pastel, Tech-Forward.
- **Visual Effects:** Glassmorphism, subtle gradients, smooth animations (Framer Motion or CSS transitions).

## Color Palette (Pastel Theme)
- **Primary:** `#427cf0` (Blue - Trust/Tech)
- **Secondary:** `#e2f9f3` (Mint Green - Growth/Speed)
- **Accent:** `#e8f0fe` (Soft Blue / Soft Coral as per wireframe)
- **Backgrounds:** 
  - Light: `#f6f6f8` (Off-white)
  - Dark: `#101622` (Slate/Navy)
- **Text:** 
  - Heading: Slate-900 (Light Mode)
  - Body: Slate-600 (Light Mode)

## Typography
- **Primary Font:** `Inter` (Sans-serif) or modern alternatives like Geist.
- **Headings:** Bold/Black weights, tight tracking (`tracking-tight`).
- **Body:** Relaxed line-height (`leading-relaxed`) for readability.

## UI Elements
- **Cards:** Rounded corners (`rounded-3xl`), Glassmorphism (`backdrop-filter: blur(12px)`), subtle border (`border-white/20`).
- **Buttons:** Large rounded corners (`rounded-xl`), bold text, shadow effects.
- **Inputs:** Clean, minimal borders, subtle focus states.
